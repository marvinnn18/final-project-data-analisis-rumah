# Tugas Besar Python Programming
Tugas Besar analisis data dan machine learning menggunakan Python untuk mempelajari pola dan memprediksi harga rumah berdasarkan karakteristik bangunan.
# Proyek Akhir Analisis Data & Machine Learning  
**Dataset: Housing Price**

# Deskripsi Proyek
Proyek ini bertujuan untuk melakukan analisis, visualisasi, serta penerapan machine learning sederhana pada dataset harga rumah.  
Analisis dilakukan untuk memahami hubungan antara karakteristik rumah (luas, jumlah kamar, dan lain-lain) terhadap harga rumah.

# Tujuan
1. Menganalisis pola dan karakteristik data harga rumah  
2. Menyajikan visualisasi data yang informatif  
3. Menerapkan supervised learning untuk prediksi harga rumah  
4. Menerapkan unsupervised learning untuk pengelompokan rumah  
5. Membuat sistem menu interaktif berbasis Python  

# Dataset
Dataset yang digunakan adalah `housingPrice.csv` dengan beberapa variabel utama:
- `area` : Luas bangunan rumah  
- `bedrooms` : Jumlah kamar tidur  
- `bathrooms` : Jumlah kamar mandi  
- `stories` : Jumlah lantai  
- `price` : Harga rumah  

Dataset disimpan pada folder:

data/housingPrice.csv


# Tools & Library
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  


# Fitur Program
Program dibuat menggunakan **sistem menu**, dengan fitur:
1. Menampilkan dataset  
2. Visualisasi data  
3. Melatih model machine learning (regresi linear)  
4. Prediksi harga rumah berdasarkan input pengguna  
5. Clustering rumah (K-Means)  
6. Pencarian rumah berdasarkan kriteria tertentu  


# Machine Learning
## Supervised Learning
- Model: Linear Regression  
- Fungsi: Memprediksi harga rumah  

## Unsupervised Learning
- Model: K-Means  
- Fungsi: Mengelompokkan rumah berdasarkan luas dan harga
